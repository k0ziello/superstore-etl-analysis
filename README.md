Superstore Analysis Pipeline (ETL)
Opis projektu
Projekt automatyzuje proces pobierania, czyszczenia i analizy danych sprzedażowych e-commerce.

Tech Stack
Python (Pandas, Matplotlib)

SQL (SQLite - Window Functions)

Data Cleaning: Normalizacja nazw kolumn, konwersja typów, obsługa brakujących danych.

Kluczowe funkcjonalności
ETL Pipeline: Automatyczne wczytywanie pliku CSV do lokalnej bazy danych SQL.

Zaawansowany SQL: Wykorzystanie funkcji okna (RANK() OVER) do wyłonienia liderów sprzedaży w kategoriach.

Wizualizacja: Generowanie wykresów rentowności produktów.

Jak uruchomić?
Zainstaluj wymagania: pip install pandas matplotlib

Uruchom skrypt: python superstore_project.py
